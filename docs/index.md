# 📎 האלגוריתם שחזר בתשובה
### *The Algorithm That Repented*

> **מה אם האלגוריתם היה מפסיק להתחבא ומתחיל להסביר את עצמו?**

---

## מה זה?

תוסף דפדפן + אפליקציית דסקטופ שיושבים איתך בזמן שאתה גולש, צופים במה שהאלגוריתם מציג לך, ו**אומרים לך את האמת**.

*"אתה רואה את זה כי 43% מהפיד שלך הוא פוליטיקה."*  
*"ביקרת ב-YouTube 94 פעמים השבוע האחרון."*  
*"השעה הכי אקטיבית שלך היא 23:00."*

האלגוריתם חזר בתשובה. הוא מתוודה. הוא מסביר.

---

## התקנה מהירה

```bash
git clone https://github.com/YOUR_USERNAME/teshuva-algorithm
cd teshuva-algorithm
npm install && npm run build
```

ואז טען את תיקיית `extension/` ב-`chrome://extensions`.

[מדריך התקנה מלא →](install.md)
