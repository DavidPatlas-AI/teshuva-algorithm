# Marketplace — שוק הסקינים וההרחבות

## מה הוא ה-Marketplace?

מקום מרכזי שבו מפתחי קהילה מפרסמים:
- **Skins** — דמויות חלופיות ל-Clippy
- **Plugins** — הרחבות שמוסיפות פונקציות
- **Themes** — ערכות עיצוב לפופ-אפ

---

## מבנה

```
marketplace/
├── index.md              — מסמך זה
├── api.js                — Client API לגישה ל-Marketplace
├── manifest-schema.json  — סכמת manifest לכל פריט
└── publisher-guide.md    — מדריך לפרסום
```

---

## קטגוריות

| קטגוריה | תיאור |
|---------|--------|
| `skins` | דמויות חלופיות |
| `plugins` | הרחבות פונקציונליות |
| `themes` | ערכות עיצוב |

---

## דירוג ותגובות

- כל פריט מקבל **דירוג 1–5** מהמשתמשים
- תגובות קצרות (עד 280 תווים)
- מינימום 10 דירוגים להצגה בעמוד הראשי
- פריטים עם 3+ שגיאות דיווח → ביקורת ידנית
